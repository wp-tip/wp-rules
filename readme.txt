== Description ==

The Business Rules WordPress Plugin allows site administrators to define conditionally executed actions based on occurring events.

Example use cases
- Build flexible content publishing workflows changes.
- Send customized mails to notify your users about important info.
- Create custom redirections, system messages, ...

== Installation ==

1. Install either from plugin directory or download it and upload from your WordPress admin panel.
2. Activate Business rules plugin.
3. You will find an admin menu item called WP Rules.

== Usage ==

1. Create new rule from WordPress admin > WP Rules.
2. Choose a name for this business rule.
3. Choose one trigger to fire this business rule with.
4. Choose some conditions that will check once this business rule is fired.
5. Choose some actions that will occur when this business rule is fired and conditions are met.
